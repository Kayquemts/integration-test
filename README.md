# api-for-testing
This repository contains a simple Spring Boot application designed as a testable API for integration purposes. It includes CRUD operations for managing vehicle data.

## Features
- **CRUD operations** for Vehicle entities.
- In-memory database (H2) for easy testing.
- Built with **Spring Boot**, **Spring Data JPA**, and **Lombok**.
- Designed for experimenting with integration testing.

## Technologies
- Java 17
- Spring Boot 3.5.0
- H2 Database
- Maven Dependency Management

This project is a small application under development to demonstrate my skills with Spring Boot and integration testing. It demonstrates my ability to build a functional API using modern tools and best practices.
